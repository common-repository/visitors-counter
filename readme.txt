It's a simple plugin to count your website's current visitor, todays hits, total hits without page refresh.

== Description ==

This is wordpress visitors counter plugins. when install this plugins autometically create a table, name wp-counter_total and created a widget. this widgets show three options.
1. current visitors
2. todays visitors
3. total visitors

and when any visitor, any place who browse your website then automaetically change your visitors number without page refresh.
